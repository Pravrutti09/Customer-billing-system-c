# Customer Billing System (C)

This project is a simple Customer Billing System developed using the C programming language. The purpose of the system is to manage basic billing operations and calculate the total cost of purchased items.

The application works as a console-based program where users can enter product details, quantity, and price to generate a bill for the customer.

## Features

Add product details  
Calculate total bill amount  
Display customer bill  
Simple menu-driven interface  

## Technologies Used

C Programming  
Basic Data Structures  
Structured Programming  

## How to Run

1. Download the project files.
2. Open the project in any C compiler such as CodeBlocks, Dev-C++, or GCC.
3. Compile and run the main C file.
4. Follow the menu options to generate the bill.

## Purpose of the Project

The project was created to understand the basics of C programming and to practice concepts like functions, loops, and structured program design while building a simple real-world application.
